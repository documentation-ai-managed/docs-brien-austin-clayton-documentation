<Update label="2024-11-05" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for teams to edit documentation simultaneously.
- Introduced customizable themes with your brand color `#3B82F6`.
- New search indexing for faster navigation across large docs sites.

## Improvements

- Enhanced MDX rendering speed by 40% for complex pages.
- Better mobile responsiveness with adaptive layouts.

## Bug Fixes

- Fixed parsing errors in nested code blocks.
- Resolved frontmatter validation issues with special characters.

</Update>

<Update label="2024-10-15" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated Mermaid diagram support via code blocks.
- Added JSX component auto-completion in the editor.

## Improvements

- Optimized image loading with lazy attributes.

## Bug Fixes

- Corrected escaping for `{` and `}` in prose text.
- Fixed heading hierarchy enforcement starting from H2.

</Update>

<Update label="2024-09-10" description="v1.0.0" tags={["breaking", "feature"]}>

## New Features

- Core MDX documentation engine with component support.
- YAML frontmatter validation and auto-generation.

## Breaking Changes

- Removed H1 support; all pages now start with H2 headings.

## Bug Fixes

- Initial stability fixes for component rendering.

</Update>
